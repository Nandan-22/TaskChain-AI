# TaskChain AI ⚡

A lightweight web app that automates startup customer support using 
AI and logs every decision permanently on the Shardeum blockchain.

## What it does
- Analyzes incoming customer messages instantly
- Classifies by Priority (High/Medium/Low) and Category (Sales/Support/General)
- Generates suggested replies automatically
- Logs every AI decision to the blockchain with a unique transaction hash

## Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Python, Flask
- AI: Custom NLP Engine (Rule-based intent detection)
- Blockchain: SHA-256 Hashing + Shardeum Smart Contract (Solidity)
